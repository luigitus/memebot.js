Memebot is a simple, yet powerful chat bot for Twitch.tv.

Enchance your chat experience with the best memes today!

Features:

  • Powerful custom commands. Create your own lists, counters, or just regular commands!
  • Points system: Reward your viewers with points for watching. You can even create custom commands that cost points to use!
  • Text-triggers: Any command can be converted to act as a text trigger, that triggers each time a key word is mentioned in chat. This can also be used to auto-purge certain keywords!
  • Higlhy customizable cooldowns: Each command can have both cooldowns on per-user-basis or global cooldowns!
