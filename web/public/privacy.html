Memebot privacy guidelines (state 3/3/2017)

Memebot allows login via Twitch.tv accounts.
In no event shall Memebot services permanently store login information on it's server (Including, but not limited to oauth tokens, account information etc.) provided by Twitch.tv apis unless one, or more of the following conditions are met:

1. The user has entered login information for use as a bot account in chat.

2. The user has been explicitly warned about the process beforehand and has agreed to saving data.

Memebot services do use cookies to locally store login information and other user information during your session. You can at any time revoke access tokens given to Memebot services at https://www.twitch.tv/settings/connections.

In no event shall The Copyright Holder, as described in the License be liable for any damange caused by leackage of data that are stored in cookies, or in working-memory of Memebot servers, nor shall The Copyright holder be liable for damange caused by so called "man-in-the-middle" attacks.
