Memebot relies on chat commands to operate, however a more user-friendly web interface is planned for the future.

For the time being you need to know some basic commands to use this bot!

  • Adding commands
    Adding commands is simple! just type:

    !command add !yourcoolcommand Hello I am a cool command

  • Removing commands
    Removing a command is just as easy:

    !command remove !yourcoolcommand

  • Editing commands
    Editing commands is a bit more tricky, if you require any help feel free to message me at any time!

    !command edit !yourcoolcommand thisIsAnExampleOption newValue

    A range of all possible options that can be edited can be found on the command's page.
